IndexFlow is a white-label SEO and content management platform 
built for digital marketing agencies, SEO agencies, content agencies, and freelancers. It's a standalone commercial SaaS product at indexflow.cloud, extracted from the Content Engine infrastructure originally built inside resto.restaurant.

What IndexFlow does:
IndexFlow gives agencies a single platform to manage all their client SEO and content work, with the ability to resell it under their own brand. An agency signs up, creates workspaces for each of their clients, and gets access to every tool they need to run content marketing and SEO campaigns -- all in one place.

Core capabilities:
Content Engine
Bulk draft generation powered by GPT-4o. Agencies create campaigns, generate multiple blog posts at once with AI, and the system enforces quality gates (word count, heading count, FAQ sections, image placeholders, duplicate detection). Images are automatically resolved from stock photo APIs (Pixabay, Pexels, Unsplash) with AI-generated SEO metadata (alt text, captions, search queries). Posts can be exported as MDX files or synced directly to CMS platforms.
CMS Integration 

Publish content directly to WordPress, Webflow, Shopify, Ghost, or Wix. Each integration formats posts with proper metadata, schema markup, and images for the target platform.


SEO Tools
Rank Tracker: Monitor keyword positions across search engines over time
Local Search Grid: Visualize local search rankings across geographic grids (powered by DataForSEO)
Google Search Console ingestion: Pull in GSC data for analysis
On-Page SEO Auditing: Crawl and audit site pages for SEO issues
Schema Markup Auto-Detection: AI classifies content into 20+ schema.org types and generates JSON-LD automatically, with manual override
Post-Processing Validator: Scans posts for correct rel attributes (dofollow internal, nofollow external), anchor text diversity, link density, image alt coverage
Cross-Post Link Builder: Indexes keywords across posts, suggests internal links, bulk auto-links, detects orphan pages, checks link health
Site Profile System
Crawl client sites to build a page inventory. The system uses this data during content generation to auto-insert internal links to existing site pages and add CTA blocks.
CRM
Full sales pipeline management. Contacts with company, email, phone, tags, source tracking. Customizable pipeline stages with color coding and position ordering. Deals with value, currency, stage assignment, contact linking, expected close dates. Board view and list view for the pipeline.
Invoices & Reports
Create and send invoices to clients with line items, tax calculations, multi-currency support (USD/EUR/GBP/AUD/CAD), and status tracking (draft/sent/paid/overdue/cancelled). Generate content reports (posts by status, category, type, word counts, schema coverage), SEO reports (page audits, scores), and revenue reports. Save report snapshots for historical comparison.
AI Widget & Voice
Embeddable chat widget with AI-powered responses. Voice mode with speech-to-text and text-to-speech. Agencies can deploy this on their clients' websites for customer interaction, booking, and support.
Twilio Integration
AI phone assistant with voice and SMS. Inbound SMS keyword processing (CANCEL, CONFIRM, STATUS, HOURS, HELP). Outbound messaging templates. Call logging and analytics.
White Label
Agencies can fully rebrand the platform as their own product. Upload their logo, set brand name, pick colors, configure custom domain, set support email. Their clients never see IndexFlow branding -- it looks like the agency's own software.
AI Provider Flexibility (BYOK)
Agencies can bring their own API keys for OpenAI, Anthropic, Google AI, Grok, Mistral, Cohere, Perplexity. Same for image banks (Unsplash, Pexels, Pixabay) and payments (Stripe, PayPal). This lets agencies control their own costs and usage.

Two dashboards: 
Super Admin Dashboard (for IndexFlow platform owners)
Manage all agencies, workspaces, content moderation, platform billing/subscriptions, MRR tracking, API key management, Twilio accounts, system health, user management, support tickets, platform settings and feature flags.
Client Dashboard (for agencies)
Content Engine, SEO tools, CRM, analytics, invoices, reports, widget management, Twilio settings, AI training/knowledge base, team invites, white label configuration, billing and usage, documentation, and support tickets.
